# T-Shirt Store E-Commerce Website
An ecommerce website for a simple tshirt store created as part of a VibeCoding 101 class.
## Features
- Product catalog featuring various colors,sizes, gender etc.
- Shopping cart functionality
- Simple and easy to navigate layout and curated images
- Checkout process flow with form validation and order confirmation
## Technologies Used
- React 18 with TypeScript for type-safe component development
- CSS3 for styling and responsive design
- Tailwind CSS for utility-first styling with custom design tokens
- TanStack Query (React Query) for server state management and caching
- LocalStorage for cart persistence across sessions
